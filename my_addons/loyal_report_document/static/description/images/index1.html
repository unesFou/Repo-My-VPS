Weighing Machine Integration

This module allows us to integrate your PLU products ( Vegetable /Fruits /Meat /Fish) related with weighing machine from your Odoo Database. The module helps to categorize the weighing products as PLU products and enable seperate barcodes for the same. Also it generates PLU.txt file, which will helps to update the product information including the change in price, to the weighing scale machine. Also the module helps to scan the weighing products in POS and add them in bill print, based on the weight and price.

Key Highlights!

  • Can use simple scanning process for PLU Products
  • Can easily configure the products and update price
  • We can set the identification number for any unit of measure according to our preference
  • Have Manual Barcode and PLU Setting option in product
  • Identifies the products in to weighing machine by using the PLU number
  • Identifies the products in to Weighing Scale Machine by using the PLU number
  • Can configure barcode and PLU number as per the requirement
  • Can generate the text document for the products under PLU
  • Can create and separate product category for Weighing Machine
  • Easy and Fast billing assured

Overview

This module is used to integrate the products that use plu such as Vegetables,Meat,Fish etc. by this we can measure them and sell them using weighing machine. Using this module we can update the prices as well. This will module will convert the price of the product according to the weight in the label from the weighing machine. This module can be used in the retail business where we have the use of weighing machine. This allows the user to easily configure the products into weighing machine also they can assign the respective barcode and plu number as per their requirement. This will also allow them to take the text file according to the price updations of the product. The generated barcodes from the weighing machine with particular quantity can be scanned easily and can do the billing. This will convert the price also according to the quantity of the product. We can set the same for different unit of measure other than weight

Configuration and Features

  • Simple steps are there to follow , to enable and work with Weighing Scale Labels and POS

01

Configuration

Additional configuration required in the product master and in the barcode nomenclature. Goto Barcode Nomenclature under inventory configuration, Here we can configure a format for the barcode. Initially we need to update a two digit number that can be set according to our comfort, but it must be set same in the weighing machine for the products with weight and uses the same set of barcodes. Every product that need to be weighed must be started with this two digit number. Here we are configuring for weight the identification number as 21 with 12 digits in it. This first two digits 21 need to be set in the weighing machine as well so that it will be identified by weighing machine as weighing products.

02

Under sales tab of a product configure the product with Available in POS and also to Weigh with Scale We can set a particular category for weighing machine products also for easy identification. This is different from the normal product category. You can provide a PLU number for the particular product

03

Provide the barcode of the product in general information as per our pattern assigned in the barcode nomenclature. Here the first two digit we must provide as the identification numbers that are also updated in weighing machine for weight. Here we have set 21 in the same place. the following Five digits denotes Barcode and next five digits to identify the weight of the product [For Example : 219999900000]

04

The Five digit Barcode will be automatically recorded to the sales tab

05

In Point of sale when you scan the product with particular quantity fetched from the weighing machine with respective barcode in it that will take the price according to the weight from the weighing machine label which will be printed along with the barcode structure we configured Here the default salesprice of the product recorded per kg is Rs.30

06

Here when you scan that particular product with a particular quantity the system identifies it with the division of that particular quantity and price

07

08

To take the text document of the plu products to update into the software of weighing machine,Go to the sales module, under the orders menu we have an option to 'Export Product'

09

Click on Export Check box and select Export products

10

A text document 'plu.txt' will be downloaded automatically. This document specifies the products under plu. Here we can see the product that we have configured with the plu number, Barcode, Product name , Rate per kg . This document is being updated in the weighing machine periodically to get updated with the price of the products after changing them in the product master

11

In the software of weighing machine we can set the path of the text document that is dowloaded from odoo. By this the text document downloaded will be automatically taken into the weghing machine. This will allow the user to easy updation of the product price into weighing machine while downloading the document itself
